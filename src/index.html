Sobre el proyecto

Esta es una Single Page Application (SPA) desarrollada como Proyecto Integrador 3, construida con HTML, CSS y JavaScript Vanilla (ES Modules). Permite conversar con personajes ficticios mediante Google Gemini AI, protegiendo la API Key a través de una Serverless Function en Vercel.

Sobre los personajes

Los personajes disponibles pertenecen a la serie animada "Family Guy" (propiedad de Fox/20th Television). Fueron adaptados para este proyecto educativo, sin fines comerciales, manteniendo su humor característico dentro de límites de contenido seguros y apropiados:

  • Peter Griffin — impulsivo, leal, amante de la cerveza y la TV.
  • Stewie Griffin — bebé genio, sarcástico y grandilocuente.
  • Brian Griffin — el perro intelectual, cínico pero de buen corazón.
  • Lois Griffin — la madre paciente y sensata de la familia.
